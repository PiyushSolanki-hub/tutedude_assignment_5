Task 1: Student Marks Dictionary

This program helps find a student's marks using their name.

What it does: It stores names and marks in a dictionary. When you enter a name, it shows the marks.

How to use: Run the code and type a name (like "Alice" or "Bob") when asked.

Task 2: List Slicing

This program demonstrates how to slice and reverse lists.

What it does:

Creates a list of numbers from 1 to 10.

Takes the first 5 numbers.

Reverses those numbers.

Prints the results.
